# ⚓ Responsive Footer (Rodapé Multissetorial)

> Um componente de rodapé (footer) robusto, totalmente responsivo e estruturado com HTML5 e CSS3, ideal para aplicações web que exigem organização de informações e captura de leads.

## 📖 Sobre o Projeto

Este projeto demonstra a criação de um **Rodapé Responsivo** projetado para ser integrado em portfólios, sites institucionais ou e-commerces. O foco principal foi a construção de uma interface que mantenha a hierarquia visual tanto em desktops quanto em dispositivos móveis.

A estrutura é dividida em zonas de interesse: informações institucionais, catálogo de serviços, formulário de inscrição e presença em redes sociais.

## ✨ Destaques e Diferenciais Técnicos

* **Seção de Conversão (Newsletter):** Implementação de um formulário de inscrição direto no rodapé, permitindo a captura de e-mails para estratégias de marketing.
* **Navegação Estruturada:** Coluna central dedicada a "Nossos Serviços", listando áreas como Web Design, UX/UI e Desenvolvimento Mobile.
* **Integração Social Media:** Conjunto completo de ícones interativos (Facebook, Instagram, Twitter, YouTube e LinkedIn) utilizando a biblioteca Font Awesome.
* **UX de Contato:** Links diretos e clicáveis para telefone e e-mail, facilitando a comunicação imediata do usuário com a marca.
* **Copyright Personalizado:** Inclusão de uma faixa inferior dedicada a direitos autorais e créditos, reforçando o profissionalismo da entrega.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e acessível do componente.
* **CSS3:** Gerenciamento de layout responsivo, tipografia e estilização de formulários.
* **Font Awesome (v5.15.2):** Iconografia vetorial para suporte visual e intuitivo.
